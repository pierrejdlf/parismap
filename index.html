paris m'appartient . fr

projet ouvert de cartographie locale en Île-de-France. (ancien site)

c'est

  • une carte en ligne
  • (peut-être bientôt) une application android et iOS
  • (offline) le compte twitter (robot) @parismap

la carte aggrège

  • les événements culturels glânés deci-delà:
    • lylo
    • quefaire
    • linternaute
    • calenda
    • cibul
    • sowprog
    • (peut-être bientôt) http://www.agendadulibre.org
    • (peut-être bientôt) http://bn.parinux.org/p/agenda
  • (offline) les messages laissés depuis l'application et les tweets géolocalisés

(offline) twittez @parismap pour écrire sur la carte

  • si votre tweet est géoréférencé, il apparaitra sur la carte à l'endroit où vous êtes
  • vous pouvez aussi utiliser la syntaxe @parismap #loc:bastille la manif prend le boulevard #voltaire, pour le géoréférencer automatiquement

(offline) écoutez la carte !

si votre tweet contient #qui ou #quoi, vous recevrez 3 réponses:

  • @parismap #qui > vous renvoie les 3 messages les plus proches
  • @parismap #quoi > vous répondra les 3 événements les plus proches
  • @parismap #metro, #bus, #velib > vous informe sur les stations les plus proches: horaires, disponibilités

il y a une très chouette vidéo qui montre tout ça.

avec les principes et objectifs suivants

  • but non lucratif
  • privilégier le local
  • simple (une seule vue, pas d'options, texte avant tout)
  • éviter la répétition
  • ouverture des données (API) et du code
  • sans publicité
  • pas de compte utilisateur, pas d'inscription obligatoire
