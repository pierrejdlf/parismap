paris m'appartient . fr

projet ouvert de cartographie locale en Île-de-France

quand le monde entier a déjà été cartographié, que reste t'il à explorer ?

en pratique, c'est

  • une carte en ligne
  • une application android et iOS
  • le compte twitter @parismap (c'est un robot)

la carte agrège

  • agendas à intégrer: http://www.agendadulibre.org/ http://bn.parinux.org/p/agenda
  • les événements culturels glânés au détour de merveilleux sites tels que ( lylo, quefaire, linternaute, calenda, cibul, sowprog, etc…). cliquez sur les marqueurs pour consulter le site producteur de l'évènement
  • les messages laissés depuis l'application et les tweets géolocalisés

twittez @parismap pour écrire sur la carte

  • si votre tweet est géoréférencé, il apparaitra sur la carte à l'endroit où vous êtes
  • vous pouvez aussi utiliser la syntaxe @parismap #loc:bastille la manif prend le boulevard #voltaire, pour le géoréférencer automatiquement

écoutez la carte !

si votre tweet contient #qui ou #quoi, vous recevrez 3 réponses - @parismap #qui > vous renvoie les 3 messages les plus proches - @parismap #quoi > vous répondra les 3 événements les plus proches

il y a une très chouette vidéo qui montre tout ça

avec les principes et objectifs suivants

  • but non lucratif
  • privilégier le local
  • simple (une seule vue, pas d'options, texte avant tout)
  • éviter la répétition
  • ouverture des données et du code (Licence LGPL)
  • sans publicité
  • pas de compte utilisateur, pas d'inscription obligatoire
    • … avec l'esprit du manifeste actif !

  • il est porté initialement par le petit collectif un peu plus tôt,
